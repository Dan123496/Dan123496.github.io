Web Design and Development

A sample of my HTML and CSS code use to create this websit.

For my Web Design and Development unit, I have been making this very website. I have created this website by learning HTML and CSS code. Throughout this unit I have learnt the basics of HTML and CSS; HTML tags, how to make HTML lists, tables, adding images, links and more. In weeks one and two, I learnt the basics of HTML and CSS respectively. In week three, how to layout a website using the CSS box method; using containers, float, absolute and relative positioning. I also learnt about fluid, fixed and relative layout option. In week four, how to add different accessibility and usability features. I then learnt how to apply media quires to my CSS to edit how the CSS is applied on different screen sizes, like mobile or tablet screens. Finally, I learnt how to add transforms, transitions and animations. I have tried to apply all I have learnt over the past few weeks to develop this website. Beside is a screenshot of some of my HTML (left) and CSS(right) code which is used to create this website. I had previously learned some basic html as part of a BTEC I completed in college, however this was very basic HTML, only coded in notepad and had no CSS. This helped me somewhat in writing my HTML code, but sadly I had to relearn most of what I had learnt previously as I could not remember all of it.
